Handling Class Imbalance Problem in Credit Card Fraud Detection II

#1010881

Class imbalance is a problem in machine learning that typically occurs when one class contains significantly fewer number samples than the other class in a training data In this project, class imbalance problem in credit card fraud data is addressed First, resampling methods like oversampling /undersampling are employed to overcome this problem Then ML algorithms are trained on balanced data to predict fraudulent transactions

High Frequency Trading of Bitcoin and other coins with Machine learning

#1010903

The features related to the bitcoin prices should be collected and stored in a database . All the bitcoin prices in the last 3 years should be stored in the database. A linear regression should be applied to see whether these features explain the prices. Then ML algorithms should be applied to predict the future values. A java application should be written to automate the buying and selling options depending on the predicted prices.
